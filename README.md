📁 File Dashboard – Frontend

> A professional, responsive file management dashboard built for Queensford College using React and Vite.

 ⚙️ Tech Stack

- React + Vite
- Axios
- CSS3
- Role-based Access (Manager & Coordinator)
- Deployed via Vercel
